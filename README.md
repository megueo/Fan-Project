RTS game based on **Castle Story (Sauropod Studio)**, made in Unity.

> the name of the game is subject to change

## How to Run?

1. Clone this project with ```git clone https://github.com/megueo/Fan-Project.git``` 
2. Open **Unity Hub**
3. Click in **Add**
4. Search the project in disk
5. Open it and enjoy :D

## Current Controls:

**W / A / S / D:** Move the camera up, left, right and back respectively\
**Press & Hold Scroll Wheel:** Move the camera angles\
**Left Mouse Button:** Select Units\
**Right Mouse Button:** Move Units

## Dev Tools:

**Slash Key (/):** Open the console\
**ESC Key or Slash Key (/) again:** Close the console\

### Current Commands: 
```clear```, ```spawn totem```
